## Description

Guten Tag is a modern, Gutenberg-first WordPress theme designed for **block-based content creation**.

It supports:

- Full Site Editing (FSE) with header, footer, and template parts.
- Responsive design for mobile, tablet, and desktop.
- Gutenberg block features including wide and full alignment.
- Custom editor color palette and typography.
- Lightweight and fast-loading for optimal performance.

Ideal for bloggers, small businesses, portfolios, and anyone who wants to build a site entirely with Gutenberg blocks.

## Installation

1. Download the `guten-tag.zip` file.
2. In your WordPress dashboard, go to **Appearance → Themes → Add New → Upload Theme**.
3. Select the `guten-tag.zip` file and click **Install Now**.
4. Activate the theme.
5. Optional: Go to **Appearance → Customize** to configure site identity, menus, and colors.
6. Start building content using Gutenberg blocks or Full Site Editing templates.

## Frequently Asked Questions

**Q: Is Guten Tag compatible with older WordPress versions?**  
A: Guten Tag requires WordPress 6.0 or newer to fully support Gutenberg and Full Site Editing.

**Q: Can I use my own logo?**  
A: Yes. You can add a custom logo via **Appearance → Customize → Site Identity**.

**Q: Is Guten Tag mobile-friendly?**  
A: Absolutely. The theme is fully responsive and adapts to mobile, tablet, and desktop screens.

**Q: Can I use this theme with page builders?**  
A: The theme is designed for Gutenberg and FSE. While some page builders may work, the best experience is with Gutenberg blocks.
